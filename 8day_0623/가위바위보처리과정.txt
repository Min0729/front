1. 반복 (  무승부 일동안 : 둘 다 같은 값일 동안 )
        1.1 사용자가 가위바위보를 합니다.
        1.2 컴퓨터가 가위바위보를 합니다.
        1.3 사용자와 컴퓨터의 가위바위보 값을  판단
             1.3.1  같은값이 아닌 경우  사용자의 승패를 판단              
                   -   사용자가 가위:  컴퓨터는 보자기
                   -   사용자가 바위 : 컴퓨터는 가위
                   -   사용자가  보 :  컴퓨터는  주먹
             1.3.2  같은값인 경우 그대로 반복을 수행함   
            
2.  사용자의 승패 정보를 출력한다.
3.  종료
